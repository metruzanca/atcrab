# Lexicons directory convention

Directory names under `src/lexicons/` must follow the pattern `<domain>_<tld>`, matching the reversed domain they represent.

## Examples

| Domain | Directory |
|---|---|
| `standard.site` | `standard_site` |
| `leaflet.pub` | `leaflet_pub` |
| `app.bsky` | `app_bsky` |
| `com.atproto` | `com_atproto` |
